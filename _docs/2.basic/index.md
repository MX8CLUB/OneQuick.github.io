OneQuick 采用符合直觉的设计，功能设计融入系统自身的框架内，降低使用门槛。  

但这仍需一点点时间熟悉。就好像熟悉`Ctrl C`为复制，`Ctrl Z`为撤销一样。  

请相信这是值得的，略微熟悉后，这些新的操作会带给您不一样的体验。
